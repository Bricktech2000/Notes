## Normal Mode

#### Motion Commands

<br><br>

|                        |              |
| ---------------------- | ------------ |
| _h_ _j_ _k_ _l_ &emsp; | (arrow keys) |
| _w_                    | word         |
| \_                     | line         |

## Normal Mode

#### Phrases

<br><br>

|                  |                  |
| ---------------- | ---------------- |
| **d**_w_         | delete word      |
| **y**2*w* &emsp; | yank two words   |
| **y**\_          | yank line        |
| **d**2\_         | delete two lines |
| $\dots$          | $\dots$          |
