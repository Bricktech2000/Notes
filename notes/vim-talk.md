## Normal Mode

### Motion Commands

<br>

|                        |              |
| ---------------------- | ------------ |
| _h_ _j_ _k_ _l_ &emsp; | (arrow keys) |
| _w_                    | word         |
| \_                     | line         |

[//]: # 'USE NUMBERS'

<br><br><br><br><br><br><br><br><br><br><br><br>

## Normal Mode

### Verb Commands

<br>

|              |             |
| ------------ | ----------- |
| **d** &emsp; | delete      |
| **y**        | yank (copy) |
| **p**        | paste       |

<br><br><br><br><br><br><br><br><br><br><br><br>

## Normal Mode

### Phrases

<br>

|                  |                  |
| ---------------- | ---------------- |
| **d**_w_         | delete word      |
| **y**2*w* &emsp; | yank two words   |
| **y**\_          | yank line        |
| **d**2\_         | delete two lines |
| $\dots$          | $\dots$          |

[//]: # 'easy to remember, switch keyboard layouts'
