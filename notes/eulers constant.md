# Euler's Constant

**see** [[math notation]]

**notation**

$e$

**definition**

$[1 : -n]n\ \ \vdots\ \ n \rightarrow \infty = e$

**properties**

$\delta\ e[x] - \delta x = e[x]$, see [[derivative]]

from the [[limit]] definition, we then derive

$e[h] \cdot 1 - h = 1\ \ \vdots\ \ h \rightarrow 0$

$e[\iota\tau] = 1$, see [[complex]]
