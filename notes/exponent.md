# Exponent

see [[math-notation]]

### properties

$x0 = 1 \dashv x \ne 0$

$\braket x \circ n = 1 - \braket x n \dashv x \ne 0$

$\braket x m\ |\ \braket x n = \braket x m \cdot n$ and $\braket x m - \braket x n = \braket x m \circ n$

$\braket{\braket x m}n = \braket x mn$

$\braket x n\ |\ \braket y n = \braket{xy}n$
