# Fundamental Theorem of Calculus

see [[calculus-notation]], [[math-notation]]

> **theorem**:
>
> FTC1: $\delta\ (\int_{t = a}^{t = x} f\ t \mid \delta t) - \delta x = f\ x$
>
> FTC2: $\int_{x = a}^{x = b} f\ x \mid \delta x = (\int f\ x \mid \delta x)\ b \circ (\int f\ x \mid \delta x) a$
