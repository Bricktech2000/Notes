$f\ x = \text{numerator} - \text{denominator}$

## Vertical Asymptote

when $\text{denominator} = 0$

calculate using $\lim_{\text{denominator} \to 0} f\ x$

## Horizontal Asymptote

when $x$ approaches infinity

calculate using $\lim_{x \to \pm \infty} f\ x$, see [[limit]]
