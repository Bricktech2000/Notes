# Positional Numeral System

see [[conventional-math-notation]], [[number]]

> **notation**:
>
> $(123)_{base}$, where
>
> - $base$ is a [[natural]]
> - $123$ is a [[number]]
>
> > **example**: $35_{10}$
>
> > **example**: $1001_{2}$
